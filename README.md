# dotfiles

Dependencies:

- nix https://nixos.org/download.html
- Home Manager https://github.com/nix-community/home-manager
- Brew

## Home Manager

```
home-manager switch
```

## Brew

```
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

brew bundle install
```
